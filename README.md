## 🔍 Overview

<a href="#popClip">PopClip</a> serves Mac power users who work extensively with text across multiple applications and who want to reduce the friction of text operations that currently require multiple steps — copying, switching applications, pasting, formatting, searching — into single-click actions at the point of selection.

The <a href="#popClip">research and writing workflow</a> benefits most from PopClip — writers who regularly look up words, search for references, translate phrases, add notes to a research collection, and perform other text operations across multiple applications find that PopClip reduces dozens of daily multi-step operations to single clicks. The actions appear where the text is, eliminating the cognitive overhead of remembering keyboard shortcuts and navigating menus. The <a href="#popClip">developer workflow</a> uses PopClip extensions for searching documentation, looking up Stack Overflow answers, running code formatting, and other developer-specific text operations — with extensions available for common developer tools and services.

The <a href="#popClip">multilingual user use case</a> benefits from instant translation extensions — selecting text in a foreign language and translating it immediately from the popup, without switching to a translation website or application. <a href="#popClip">Setapp availability</a> includes PopClip in the Setapp Mac application subscription — Setapp subscribers access PopClip without additional purchase as part of the collection.

<a href="#popClip">Custom extension creation</a> enables technically inclined users to build personal PopClip extensions — writing custom actions in JavaScript, shell scripts, or AppleScript for personal workflow needs that the existing extension library doesn't cover. <a href="#popClip">PopClip for iPad</a> brings the same text selection action bar to iPad with a similar extension ecosystem.

<a href="#popClip">Privacy-conscious design</a> processes all text actions locally where possible — actions that don't require external services (copy, cut, change case, count words) never leave the Mac. <a href="#popClip">Pilotmoon Software development</a> maintains PopClip with regular updates, extension library curation, and active engagement with the user community that contributes extensions and feedback.

## 💬 FAQ

<details>
<summary>How do I install PopClip extensions?</summary>

Extensions install from the PopClip extension library on the PopClip website or through in-app extension discovery. Many extensions install with a single click.

</details>

<details>
<summary>Can I control which extensions appear in the PopClip bar?</summary>

Yes. Installed extensions are configurable to show or hide in the popup, keeping the bar compact and relevant to individual workflow needs.

</details>

<details>
<summary>Does PopClip support keyboard shortcut triggering?</summary>

Yes. A configurable keyboard shortcut activates PopClip when the automatic selection popup isn't triggered.

</details>

<details>
<summary>Does PopClip run on Apple Silicon?</summary>

Yes. Current versions support Apple Silicon Macs natively.

</details>

<details>
<summary>How many extensions are available for PopClip?</summary>

Hundreds of extensions are available covering a very wide range of applications and services. The extension library continues growing through community contributions.

</details>

<details>
<summary>What is the pricing for PopClip?</summary>

PopClip is available as a paid purchase from the Mac App Store and through the Setapp subscription. Check current pricing on the PopClip website or Mac App Store.

</details>
